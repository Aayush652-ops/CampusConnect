# CampusConnect

CampusConnect is a centralized web platform designed to help students discover campus events, communities, internships, competitions, and opportunities from a single interface.

## Developer Role

- Designed a centralized web platform enabling students to discover events, communities, internships, competitions, and opportunities from a single interface.
- Conducted user research, developed personas, mapped user journeys, and created wireframes and high-fidelity prototypes to improve opportunity discovery.
- Designed personalized dashboards and information architecture that reduced friction caused by fragmented communication channels.
- Applied design thinking methodologies from problem identification and research to prototyping, validation, and user experience optimization.

## Technologies / Tools Used

- Figma
- FigJam
- User Research
- Wireframing
- Prototyping
- Information Architecture

## Project Structure

- `web/` – static web platform prototype
- `Design/` – UX research, user journey, and product design overview

## How to use

1. Open `CampusConnect/web/index.html` in a browser to view the prototype.
2. Review the design documentation in `CampusConnect/Design/UX-Overview.md`.
3. Use `CampusConnect/web/styles.css` and `CampusConnect/web/app.js` as the starting point for interactive UI development.
